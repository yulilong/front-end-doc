# CSS3ж–°еұһжҖ§д»Ӣз»Қ

# 1. иҫ№жЎҶзӣёе…і

еұһжҖ§ жҸҸиҝ° CSS
border-image и®ҫзҪ®жүҖжңү border-image-* еұһжҖ§зҡ„з®ҖеҶҷеұһжҖ§гҖӮи®ҫзҪ®иҫ№жЎҶеӣҫзүҮ 3
border-radius и®ҫзҪ®жүҖжңүеӣӣдёӘ border-*-radius еұһжҖ§зҡ„з®ҖеҶҷеұһжҖ§гҖӮ 3
box-shadow еҗ‘ж–№жЎҶж·»еҠ дёҖдёӘжҲ–еӨҡдёӘйҳҙеҪұгҖӮ 3

border-image е…·дҪ“еұһжҖ§

еҖј жҸҸиҝ°
border-image-source з”ЁдәҺжҢҮе®ҡиҰҒз”ЁдәҺз»ҳеҲ¶иҫ№жЎҶзҡ„еӣҫеғҸзҡ„дҪҚзҪ®
border-image-slice еӣҫеғҸиҫ№з•Ңеҗ‘еҶ…еҒҸз§»
border-image-width еӣҫеғҸиҫ№з•Ңзҡ„е®ҪеәҰ
border-image-outset з”ЁдәҺжҢҮе®ҡеңЁиҫ№жЎҶеӨ–йғЁз»ҳеҲ¶ border-image-area зҡ„йҮҸ
border-image-repeat з”ЁдәҺи®ҫзҪ®еӣҫеғҸиҫ№з•ҢжҳҜеҗҰеә”йҮҚеӨҚпјҲrepeatпјүгҖҒжӢүдјёпјҲstretchпјүжҲ–й“әж»ЎпјҲroundпјүгҖӮ

border-radiusеңҶи§’еұһжҖ§

еұһжҖ§ жҸҸиҝ°
border-radius жүҖжңүеӣӣдёӘиҫ№и§’ border---radius еұһжҖ§зҡ„зј©еҶҷ
border-top-left-radius е®ҡд№үдәҶе·ҰдёҠи§’зҡ„еј§еәҰ
border-top-right-radius е®ҡд№үдәҶеҸідёҠи§’зҡ„еј§еәҰ
border-bottom-right-radius е®ҡд№үдәҶеҸідёӢи§’зҡ„еј§еәҰ
border-bottom-left-radius е®ҡд№үдәҶе·ҰдёӢи§’зҡ„еј§еәҰ

}

# 4. ж–Үжң¬ж•Ҳжһң

еұһжҖ§ жҸҸиҝ° CSS
hanging-punctuation и§„е®ҡж ҮзӮ№еӯ—з¬ҰжҳҜеҗҰдҪҚдәҺзәҝжЎҶд№ӢеӨ–гҖӮ 3
punctuation-trim и§„е®ҡжҳҜеҗҰеҜ№ж ҮзӮ№еӯ—з¬ҰиҝӣиЎҢдҝ®еүӘгҖӮ 3
text-align-last и®ҫзҪ®еҰӮдҪ•еҜ№йҪҗжңҖеҗҺдёҖиЎҢжҲ–зҙ§жҢЁзқҖејәеҲ¶жҚўиЎҢз¬Ұд№ӢеүҚзҡ„иЎҢгҖӮ 3
text-emphasis еҗ‘е…ғзҙ зҡ„ж–Үжң¬еә”з”ЁйҮҚзӮ№ж Үи®°д»ҘеҸҠйҮҚзӮ№ж Үи®°зҡ„еүҚжҷҜиүІгҖӮ 3
text-justify и§„е®ҡеҪ“ text-align и®ҫзҪ®дёә "justify" ж—¶жүҖдҪҝз”Ёзҡ„еҜ№йҪҗж–№жі•гҖӮ 3
text-outline и§„е®ҡж–Үжң¬зҡ„иҪ®е»“гҖӮ 3
text-overflow и§„е®ҡеҪ“ж–Үжң¬жәўеҮәеҢ…еҗ«е…ғзҙ ж—¶еҸ‘з”ҹзҡ„дәӢжғ…гҖӮ 3
text-shadow еҗ‘ж–Үжң¬ж·»еҠ йҳҙеҪұгҖӮ 3
text-wrap и§„е®ҡж–Үжң¬зҡ„жҚўиЎҢи§„еҲҷгҖӮ 3
word-break и§„е®ҡйқһдёӯж—Ҙйҹ©ж–Үжң¬зҡ„жҚўиЎҢи§„еҲҷгҖӮ 3
word-wrap е…Ғи®ёеҜ№й•ҝзҡ„дёҚеҸҜеҲҶеүІзҡ„еҚ•иҜҚиҝӣиЎҢеҲҶеүІе№¶жҚўиЎҢеҲ°дёӢдёҖиЎҢгҖӮ 3

# 5. еӯ—дҪ“

еңЁ CSS3 д№ӢеүҚпјҢweb и®ҫи®ЎеёҲеҝ…йЎ»дҪҝз”Ёе·ІеңЁз”ЁжҲ·и®Ўз®—жңәдёҠе®үиЈ…еҘҪзҡ„еӯ—дҪ“гҖӮйҖҡиҝҮ CSS3пјҢweb и®ҫи®ЎеёҲеҸҜд»ҘдҪҝз”Ёд»–д»¬е–ңж¬ўзҡ„д»»ж„Ҹеӯ—дҪ“гҖӮ

FirefoxгҖҒChromeгҖҒSafari д»ҘеҸҠ Opera ж”ҜжҢҒ .ttf (True Type Fonts) е’Ң .otf (OpenType Fonts) зұ»еһӢзҡ„еӯ—дҪ“гҖӮ

Internet Explorer 9+ ж”ҜжҢҒж–°зҡ„ @font-face и§„еҲҷпјҢдҪҶжҳҜд»…ж”ҜжҢҒ .eot зұ»еһӢзҡ„еӯ—дҪ“ (Embedded OpenType)гҖӮ

**жіЁйҮҠпјҡ**Internet Explorer 8 д»ҘеҸҠжӣҙж—©зҡ„зүҲжң¬дёҚж”ҜжҢҒж–°зҡ„ @font-face и§„еҲҷгҖӮ

@font-face {
    font-family: myFirstFont;
    src: url(sansation_bold.woff);
    font-weight:bold;
}

дёӢйқўзҡ„иЎЁж јеҲ—еҮәдәҶиғҪеӨҹеңЁ @font-face и§„еҲҷдёӯе®ҡд№үзҡ„жүҖжңүеӯ—дҪ“жҸҸиҝ°з¬Ұпјҡ

жҸҸиҝ°з¬Ұ еҖј жҸҸиҝ°
font-family name еҝ…йңҖгҖӮи§„е®ҡеӯ—дҪ“зҡ„еҗҚз§°гҖӮ
src URL еҝ…йңҖгҖӮе®ҡд№үеӯ—дҪ“ж–Үд»¶зҡ„ URLгҖӮ
font-stretch normalпјҢcondensedпјҢultra-condensedпјҢextra-condensedпјҢsemi-condensedпјҢexpandedпјҢsemi-expandedпјҢextra-expandedпјҢultra-expanded еҸҜйҖүгҖӮе®ҡд№үеҰӮдҪ•жӢүдјёеӯ—дҪ“гҖӮй»ҳи®ӨжҳҜ "normal"гҖӮ
font-style normalпјҢitalicпјҢoblique еҸҜйҖүгҖӮе®ҡд№үеӯ—дҪ“зҡ„ж ·ејҸгҖӮй»ҳи®ӨжҳҜ "normal"гҖӮ
font-weight normalпјҢboldпјҢ100пјҢ200пјҢ300пјҢ400пјҢ500пјҢ600пјҢ700пјҢ800пјҢ900 еҸҜйҖүгҖӮе®ҡд№үеӯ—дҪ“зҡ„зІ—з»ҶгҖӮй»ҳи®ӨжҳҜ "normal"гҖӮ
unicode-range unicode-range еҸҜйҖүгҖӮе®ҡд№үеӯ—дҪ“ж”ҜжҢҒзҡ„ UNICODE еӯ—з¬ҰиҢғеӣҙгҖӮй»ҳи®ӨжҳҜ "U+0-10FFFF"гҖӮ

йҖҡиҝҮ CSS3пјҢweb и®ҫи®ЎеёҲеҸҜд»ҘдҪҝз”Ёд»–д»¬е–ңж¬ўзҡ„д»»ж„Ҹеӯ—дҪ“гҖӮ

# 6. 2DиҪ¬жҚўtransform

Property жҸҸиҝ° CSS
transform йҖӮз”ЁдәҺ2DжҲ–3DиҪ¬жҚўзҡ„е…ғзҙ  3
transform-origin е…Ғи®ёжӮЁжӣҙж”№иҪ¬еҢ–е…ғзҙ дҪҚзҪ® 3

2D иҪ¬жҚўж–№жі•

еҮҪж•° жҸҸиҝ°
matrix(n,n,n,n,n,n) е®ҡд№ү 2D иҪ¬жҚўпјҢдҪҝз”Ёе…ӯдёӘеҖјзҡ„зҹ©йҳөгҖӮеҢ…еҗ«ж—ӢиҪ¬пјҢзј©ж”ҫпјҢз§»еҠЁпјҲе№із§»пјүе’ҢеҖҫж–ңеҠҹиғҪгҖӮ
translate(x,y) е®ҡд№ү 2D иҪ¬жҚўпјҢжІҝзқҖ X е’Ң Y иҪҙз§»еҠЁе…ғзҙ гҖӮ
translateX(n) е®ҡд№ү 2D иҪ¬жҚўпјҢжІҝзқҖ X иҪҙз§»еҠЁе…ғзҙ гҖӮ
translateY(n) е®ҡд№ү 2D иҪ¬жҚўпјҢжІҝзқҖ Y иҪҙз§»еҠЁе…ғзҙ гҖӮ
scale(x,y) е®ҡд№ү 2D зј©ж”ҫиҪ¬жҚўпјҢж”№еҸҳе…ғзҙ зҡ„е®ҪеәҰе’Ңй«ҳеәҰгҖӮ
scaleX(n) е®ҡд№ү 2D зј©ж”ҫиҪ¬жҚўпјҢж”№еҸҳе…ғзҙ зҡ„е®ҪеәҰгҖӮ
scaleY(n) е®ҡд№ү 2D зј©ж”ҫиҪ¬жҚўпјҢж”№еҸҳе…ғзҙ зҡ„й«ҳеәҰгҖӮ
rotate(angle) е®ҡд№ү 2D ж—ӢиҪ¬пјҢеңЁеҸӮж•°дёӯи§„е®ҡи§’еәҰгҖӮ
skew(x-angle,y-angle) е®ҡд№ү 2D еҖҫж–ңиҪ¬жҚўпјҢжІҝзқҖ X е’Ң Y иҪҙгҖӮ
skewX(angle) е®ҡд№ү 2D еҖҫж–ңиҪ¬жҚўпјҢжІҝзқҖ X иҪҙгҖӮ

scaleY(y) е®ҡд№ү 3D зј©ж”ҫиҪ¬жҚўпјҢйҖҡиҝҮз»ҷе®ҡдёҖдёӘ Y иҪҙзҡ„еҖјгҖӮ
scaleZ(z) е®ҡд№ү 3D зј©ж”ҫиҪ¬жҚўпјҢйҖҡиҝҮз»ҷе®ҡдёҖдёӘ Z иҪҙзҡ„еҖјгҖӮ

# 8. иҝҮжёЎtransition

дёӢиЎЁеҲ—еҮәдәҶжүҖжңүзҡ„иҝҮжёЎеұһжҖ§:

еұһжҖ§ жҸҸиҝ° CSS
transition з®ҖеҶҷеұһжҖ§пјҢз”ЁдәҺеңЁдёҖдёӘеұһжҖ§дёӯи®ҫзҪ®еӣӣдёӘиҝҮжёЎеұһжҖ§гҖӮ 3
transition-property и§„е®ҡеә”з”ЁиҝҮжёЎзҡ„ CSS еұһжҖ§зҡ„еҗҚз§°гҖӮ 3
transition-duration е®ҡд№үиҝҮжёЎж•ҲжһңиҠұиҙ№зҡ„ж—¶й—ҙгҖӮй»ҳи®ӨжҳҜ 0гҖӮ 3
transition-timing-function и§„е®ҡиҝҮжёЎж•Ҳжһңзҡ„ж—¶й—ҙжӣІзәҝгҖӮй»ҳи®ӨжҳҜ "ease"гҖӮ 3
transition-delay и§„е®ҡиҝҮжёЎж•ҲжһңдҪ•ж—¶ејҖе§ӢгҖӮй»ҳи®ӨжҳҜ 0гҖӮ 3

# 9. еҠЁз”»animation

дёӢйқўзҡ„иЎЁж јеҲ—еҮәдәҶ @keyframes и§„еҲҷе’ҢжүҖжңүеҠЁз”»еұһжҖ§пјҡ

еұһжҖ§ жҸҸиҝ° CSS
@keyframes и§„е®ҡеҠЁз”»гҖӮ 3
animation жүҖжңүеҠЁз”»еұһжҖ§зҡ„з®ҖеҶҷеұһжҖ§пјҢйҷӨдәҶ animation-play-state еұһжҖ§гҖӮ 3
animation-name и§„е®ҡ @keyframes еҠЁз”»зҡ„еҗҚз§°гҖӮ 3
animation-duration и§„е®ҡеҠЁз”»е®ҢжҲҗдёҖдёӘе‘ЁжңҹжүҖиҠұиҙ№зҡ„з§’жҲ–жҜ«з§’гҖӮй»ҳи®ӨжҳҜ 0гҖӮ 3
animation-timing-function и§„е®ҡеҠЁз”»зҡ„йҖҹеәҰжӣІзәҝгҖӮй»ҳи®ӨжҳҜ "ease"гҖӮ 3
animation-fill-mode и§„е®ҡеҪ“еҠЁз”»дёҚж’ӯж”ҫж—¶пјҲеҪ“еҠЁз”»е®ҢжҲҗж—¶пјҢжҲ–еҪ“еҠЁз”»жңүдёҖдёӘе»¶иҝҹжңӘејҖе§Ӣж’ӯж”ҫж—¶пјүпјҢиҰҒеә”з”ЁеҲ°е…ғзҙ зҡ„ж ·ејҸгҖӮ 3
animation-delay и§„е®ҡеҠЁз”»дҪ•ж—¶ејҖе§ӢгҖӮй»ҳи®ӨжҳҜ 0гҖӮ 3
animation-iteration-count и§„е®ҡеҠЁз”»иў«ж’ӯж”ҫзҡ„ж¬Ўж•°гҖӮй»ҳи®ӨжҳҜ 1гҖӮ 3
animation-direction и§„е®ҡеҠЁз”»жҳҜеҗҰеңЁдёӢдёҖе‘ЁжңҹйҖҶеҗ‘ең°ж’ӯж”ҫгҖӮй»ҳи®ӨжҳҜ "normal"гҖӮ 3
animation-play-state и§„е®ҡеҠЁз”»жҳҜеҗҰжӯЈеңЁиҝҗиЎҢжҲ–жҡӮеҒңгҖӮй»ҳи®ӨжҳҜ "running"гҖӮ 3

# 10. ж–Үжң¬еӨҡеҲ—еұһжҖ§

йҖҡиҝҮ CSS3пјҢжӮЁиғҪеӨҹеҲӣе»әеӨҡдёӘеҲ—жқҘеҜ№ж–Үжң¬иҝӣиЎҢеёғеұҖ - е°ұеғҸжҠҘзәёйӮЈж ·пјҒ

еұһжҖ§ жҸҸиҝ°
column-count жҢҮе®ҡе…ғзҙ еә”иҜҘиў«еҲҶеүІзҡ„еҲ—ж•°гҖӮ
column-fill жҢҮе®ҡеҰӮдҪ•еЎ«е……еҲ—
column-gap жҢҮе®ҡеҲ—дёҺеҲ—д№Ӣй—ҙзҡ„й—ҙйҡҷ
column-rule жүҖжңү column-rule-* еұһжҖ§зҡ„з®ҖеҶҷ
column-rule-color жҢҮе®ҡдёӨеҲ—й—ҙиҫ№жЎҶзҡ„йўңиүІ
column-rule-style жҢҮе®ҡдёӨеҲ—й—ҙиҫ№жЎҶзҡ„ж ·ејҸ
column-rule-width жҢҮе®ҡдёӨеҲ—й—ҙиҫ№жЎҶзҡ„еҺҡеәҰ
column-span жҢҮе®ҡе…ғзҙ иҰҒи·Ёи¶ҠеӨҡе°‘еҲ—
column-width жҢҮе®ҡеҲ—зҡ„е®ҪеәҰ
columns и®ҫзҪ® column-width е’Ң column-count зҡ„з®ҖеҶҷ

# 11. з”ЁжҲ·з•Ңйқў

еңЁ CSS3 дёӯ, еўһеҠ дәҶдёҖдәӣж–°зҡ„з”ЁжҲ·з•Ңйқўзү№жҖ§жқҘи°ғж•ҙе…ғзҙ е°әеҜёпјҢжЎҶе°әеҜёе’ҢеӨ–иҫ№жЎҶгҖӮ

еұһжҖ§ иҜҙжҳҺ CSS
appearance е…Ғи®ёжӮЁдҪҝдёҖдёӘе…ғзҙ зҡ„еӨ–и§ӮеғҸдёҖдёӘж ҮеҮҶзҡ„з”ЁжҲ·з•Ңйқўе…ғзҙ  3
box-sizing и®ҫзҪ®зӣ’жЁЎеһӢпјҢIEзӣ’жЁЎеһӢгҖҒW3Cж ҮеҮҶзӣ’жЁЎеһӢ 3
icon дёәеҲӣдҪңиҖ…жҸҗдҫӣдәҶе°Ҷе…ғзҙ и®ҫзҪ®дёәеӣҫж Үзӯүд»·зү©зҡ„иғҪеҠӣгҖӮ 3
nav-down жҢҮе®ҡеңЁдҪ•еӨ„дҪҝз”Ёз®ӯеӨҙеҗ‘дёӢеҜјиҲӘй”®ж—¶иҝӣиЎҢеҜјиҲӘ 3
nav-index жҢҮе®ҡдёҖдёӘе…ғзҙ зҡ„Tabзҡ„йЎәеәҸ 3
nav-left жҢҮе®ҡеңЁдҪ•еӨ„дҪҝз”Ёе·Ұдҫ§зҡ„з®ӯеӨҙеҜјиҲӘй”®иҝӣиЎҢеҜјиҲӘ 3
nav-right жҢҮе®ҡеңЁдҪ•еӨ„дҪҝз”ЁеҸідҫ§зҡ„з®ӯеӨҙеҜјиҲӘй”®иҝӣиЎҢеҜјиҲӘ 3
nav-up жҢҮе®ҡеңЁдҪ•еӨ„дҪҝз”Ёз®ӯеӨҙеҗ‘дёҠеҜјиҲӘй”®ж—¶иҝӣиЎҢеҜјиҲӘ 3
outline-offset еӨ–иҪ®е»“дҝ®йҘ°е№¶з»ҳеҲ¶и¶…еҮәиҫ№жЎҶзҡ„иҫ№зјҳ 3
resize жҢҮе®ҡдёҖдёӘе…ғзҙ жҳҜеҗҰжҳҜз”ұз”ЁжҲ·и°ғж•ҙеӨ§е°Ҹ 3

# cookieгҖҒsessionгҖҒWebStorageеҢәеҲ«

дё»иҰҒеҸӮиҖғпјҡhttps://www.cnblogs.com/zr123/p/8086525.html

# 1. cookieе’Ңsession

cookieе’ҢsessionйғҪжҳҜз”ЁжқҘи·ҹиёӘжөҸи§ҲеҷЁз”ЁжҲ·иә«д»Ҫзҡ„дјҡиҜқж–№ејҸгҖӮ

еҢәеҲ«пјҡ

1гҖҒдҝқжҢҒзҠ¶жҖҒпјҡcookieдҝқеӯҳеңЁжөҸи§ҲеҷЁз«ҜпјҢsessionдҝқеӯҳеңЁжңҚеҠЎеҷЁз«Ҝ

2гҖҒдҪҝз”Ёж–№ејҸпјҡ

пјҲ1пјүcookieжңәеҲ¶пјҡеҰӮжһңдёҚеңЁжөҸи§ҲеҷЁдёӯи®ҫзҪ®иҝҮжңҹж—¶й—ҙпјҢcookieиў«дҝқеӯҳеңЁеҶ…еӯҳдёӯпјҢз”ҹе‘Ҫе‘ЁжңҹйҡҸжөҸи§ҲеҷЁзҡ„е…ій—ӯиҖҢз»“жқҹпјҢиҝҷз§Қcookieз®Җз§°дјҡиҜқcookieгҖӮеҰӮжһңеңЁжөҸи§ҲеҷЁдёӯи®ҫзҪ®дәҶcookieзҡ„иҝҮжңҹж—¶й—ҙпјҢcookieиў«дҝқеӯҳеңЁзЎ¬зӣҳдёӯпјҢе…ій—ӯжөҸи§ҲеҷЁеҗҺпјҢcookieж•°жҚ®д»Қз„¶еӯҳеңЁпјҢзӣҙеҲ°иҝҮжңҹж—¶й—ҙз»“жқҹжүҚж¶ҲеӨұгҖӮ

вҖӢ CookieжҳҜжңҚеҠЎеҷЁеҸ‘з»ҷе®ўжҲ·з«Ҝзҡ„зү№ж®ҠдҝЎжҒҜпјҢcookieжҳҜд»Ҙж–Үжң¬зҡ„ж–№ејҸдҝқеӯҳеңЁе®ўжҲ·з«ҜпјҢжҜҸж¬ЎиҜ·жұӮж—¶йғҪеёҰдёҠе®ғ

пјҲ2пјүsessionжңәеҲ¶пјҡеҪ“жңҚеҠЎеҷЁж”¶еҲ°иҜ·жұӮйңҖиҰҒеҲӣе»әsessionеҜ№иұЎж—¶пјҢйҰ–е…ҲдјҡжЈҖжҹҘе®ўжҲ·з«ҜиҜ·жұӮдёӯжҳҜеҗҰеҢ…еҗ«sessionidгҖӮеҰӮжһңжңүsessionidпјҢжңҚеҠЎеҷЁе°Ҷж №жҚ®иҜҘidиҝ”еӣһеҜ№еә”sessionеҜ№иұЎгҖӮеҰӮжһңе®ўжҲ·з«ҜиҜ·жұӮдёӯжІЎжңүsessionidпјҢжңҚеҠЎеҷЁдјҡеҲӣе»әж–°зҡ„sessionеҜ№иұЎпјҢе№¶жҠҠsessionidеңЁжң¬ж¬Ўе“Қеә”дёӯиҝ”еӣһз»ҷе®ўжҲ·з«ҜгҖӮйҖҡеёёдҪҝз”Ёcookieж–№ејҸеӯҳеӮЁsessionidеҲ°е®ўжҲ·з«ҜпјҢеңЁдәӨдә’дёӯжөҸи§ҲеҷЁжҢүз…§и§„еҲҷе°ҶsessionidеҸ‘йҖҒз»ҷжңҚеҠЎеҷЁгҖӮеҰӮжһңз”ЁжҲ·зҰҒз”ЁcookieпјҢеҲҷиҰҒдҪҝз”ЁURLйҮҚеҶҷпјҢеҸҜд»ҘйҖҡиҝҮresponse.encodeURL(url) иҝӣиЎҢе®һзҺ°пјӣAPIеҜ№encodeURLзҡ„з»“жқҹдёәпјҢеҪ“жөҸи§ҲеҷЁж”ҜжҢҒCookieж—¶пјҢurlдёҚеҒҡд»»дҪ•еӨ„зҗҶпјӣеҪ“жөҸи§ҲеҷЁдёҚж”ҜжҢҒCookieзҡ„ж—¶еҖҷпјҢе°ҶдјҡйҮҚеҶҷURLе°ҶSessionIDжӢјжҺҘеҲ°и®ҝй—®ең°еқҖеҗҺгҖӮ

3гҖҒеӯҳеӮЁеҶ…е®№пјҡcookieеҸӘиғҪдҝқеӯҳеӯ—з¬ҰдёІзұ»еһӢпјҢд»Ҙж–Үжң¬зҡ„ж–№ејҸпјӣsessionйҖҡиҝҮзұ»дјјдёҺHashtableзҡ„ж•°жҚ®з»“жһ„жқҘдҝқеӯҳпјҢиғҪж”ҜжҢҒд»»дҪ•зұ»еһӢзҡ„еҜ№иұЎ(sessionдёӯеҸҜеҗ«жңүеӨҡдёӘеҜ№иұЎ)

4гҖҒеӯҳеӮЁзҡ„еӨ§е°ҸпјҡcookieпјҡеҚ•дёӘcookieдҝқеӯҳзҡ„ж•°жҚ®дёҚиғҪи¶…иҝҮ4kbпјӣsessionеӨ§е°ҸжІЎжңүйҷҗеҲ¶гҖӮ

5гҖҒе®үе…ЁжҖ§пјҡcookieпјҡй’ҲеҜ№cookieжүҖеӯҳеңЁзҡ„ж”»еҮ»пјҡCookieж¬әйӘ—пјҢCookieжҲӘиҺ·пјӣsessionзҡ„е®үе…ЁжҖ§еӨ§дәҺcookieгҖӮ

еҺҹеӣ еҰӮдёӢпјҡпјҲ1пјүsessionIDеӯҳеӮЁеңЁcookieдёӯпјҢиӢҘиҰҒж”»з ҙsessionйҰ–е…ҲиҰҒж”»з ҙcookieпјӣ

пјҲ2пјүsessionIDжҳҜиҰҒжңүдәәзҷ»еҪ•пјҢжҲ–иҖ…еҗҜеҠЁsession_startжүҚдјҡжңүпјҢжүҖд»Ҙж”»з ҙcookieд№ҹдёҚдёҖе®ҡиғҪеҫ—еҲ°sessionIDпјӣ

пјҲ3пјүз¬¬дәҢж¬ЎеҗҜеҠЁsession_startеҗҺпјҢеүҚдёҖж¬Ўзҡ„sessionIDе°ұжҳҜеӨұж•ҲдәҶпјҢsessionиҝҮжңҹеҗҺпјҢsessionIDд№ҹйҡҸд№ӢеӨұж•ҲгҖӮ

пјҲ4пјүsessionIDжҳҜеҠ еҜҶзҡ„

пјҲ5пјүз»јдёҠжүҖиҝ°пјҢж”»еҮ»иҖ…еҝ…йЎ»еңЁзҹӯж—¶й—ҙеҶ…ж”»з ҙеҠ еҜҶзҡ„sessionIDпјҢиҝҷеҫҲйҡҫгҖӮ

6гҖҒеә”з”ЁеңәжҷҜпјҡ

cookieпјҡ

пјҲ1пјүеҲӨж–ӯз”ЁжҲ·жҳҜеҗҰзҷ»йҷҶиҝҮзҪ‘з«ҷпјҢд»ҘдҫҝдёӢж¬Ўзҷ»еҪ•ж—¶иғҪеӨҹе®һзҺ°иҮӘеҠЁзҷ»еҪ•пјҲжҲ–иҖ…и®°дҪҸеҜҶз ҒпјүгҖӮеҰӮжһңжҲ‘д»¬еҲ йҷӨcookieпјҢеҲҷжҜҸж¬Ўзҷ»еҪ•еҝ…йЎ»д»Һж–°еЎ«еҶҷзҷ»еҪ•зҡ„зӣёе…ідҝЎжҒҜгҖӮ

пјҲ2пјүдҝқеӯҳдёҠж¬Ўзҷ»еҪ•зҡ„ж—¶й—ҙзӯүдҝЎжҒҜгҖӮ

пјҲ3пјүдҝқеӯҳдёҠж¬ЎжҹҘзңӢзҡ„йЎөйқў

пјҲ4пјүжөҸи§Ҳи®Ўж•°

sessionпјҡ

Sessionз”ЁдәҺдҝқеӯҳжҜҸдёӘз”ЁжҲ·зҡ„дё“з”ЁдҝЎжҒҜпјҢеҸҳйҮҸзҡ„еҖјдҝқеӯҳеңЁжңҚеҠЎеҷЁз«ҜпјҢйҖҡиҝҮSessionIDжқҘеҢәеҲҶдёҚеҗҢзҡ„е®ўжҲ·гҖӮ

пјҲ1пјүзҪ‘дёҠе•ҶеҹҺдёӯзҡ„иҙӯзү©иҪҰ

пјҲ2пјүдҝқеӯҳз”ЁжҲ·зҷ»еҪ•дҝЎжҒҜ

пјҲ3пјүе°Ҷжҹҗдәӣж•°жҚ®ж”ҫе…ҘsessionдёӯпјҢдҫӣеҗҢдёҖз”ЁжҲ·зҡ„дёҚеҗҢйЎөйқўдҪҝз”Ё

пјҲ4пјүйҳІжӯўз”ЁжҲ·йқһжі•зҷ»еҪ•

7гҖҒзјәзӮ№пјҡ

cookieпјҡ

пјҲ1пјүеӨ§е°ҸеҸ—йҷҗ

пјҲ2пјүз”ЁжҲ·еҸҜд»Ҙж“ҚдҪңпјҲзҰҒз”ЁпјүcookieпјҢдҪҝеҠҹиғҪеҸ—йҷҗ

пјҲ3пјүе®үе…ЁжҖ§иҫғдҪҺ

пјҲ4пјүжңүдәӣзҠ¶жҖҒдёҚеҸҜиғҪдҝқеӯҳеңЁе®ўжҲ·з«ҜгҖӮ

пјҲ5пјүжҜҸж¬Ўи®ҝй—®йғҪиҰҒдј йҖҒcookieз»ҷжңҚеҠЎеҷЁпјҢжөӘиҙ№еёҰе®ҪгҖӮ

пјҲ6пјүcookieж•°жҚ®жңүи·Ҝеҫ„пјҲpathпјүзҡ„жҰӮеҝөпјҢеҸҜд»ҘйҷҗеҲ¶cookieеҸӘеұһдәҺжҹҗдёӘи·Ҝеҫ„дёӢгҖӮ

sessionпјҡ

пјҲ1пјүSessionдҝқеӯҳзҡ„дёңиҘҝи¶ҠеӨҡпјҢе°ұи¶ҠеҚ з”ЁжңҚеҠЎеҷЁеҶ…еӯҳпјҢеҜ№дәҺз”ЁжҲ·еңЁзәҝдәәж•°иҫғеӨҡзҡ„зҪ‘з«ҷпјҢжңҚеҠЎеҷЁзҡ„еҶ…еӯҳеҺӢеҠӣдјҡжҜ”иҫғеӨ§гҖӮ

пјҲ2пјүдҫқиө–дәҺcookieпјҲsessionIDдҝқеӯҳеңЁcookieпјүпјҢеҰӮжһңзҰҒз”ЁcookieпјҢеҲҷиҰҒдҪҝз”ЁURLйҮҚеҶҷпјҢдёҚе®үе…Ё

пјҲ3пјүеҲӣе»әSessionеҸҳйҮҸжңүеҫҲеӨ§зҡ„йҡҸж„ҸжҖ§пјҢеҸҜйҡҸж—¶и°ғз”ЁпјҢдёҚйңҖиҰҒејҖеҸ‘иҖ…еҒҡзІҫзЎ®ең°еӨ„зҗҶпјҢжүҖд»ҘпјҢиҝҮеәҰдҪҝз”ЁsessionеҸҳйҮҸе°ҶдјҡеҜјиҮҙд»Јз ҒдёҚеҸҜиҜ»иҖҢдё”дёҚеҘҪз»ҙжҠӨгҖӮ

# 2. WebStorage

WebStorageзҡ„зӣ®зҡ„жҳҜе…ӢжңҚз”ұcookieжүҖеёҰжқҘзҡ„дёҖдәӣйҷҗеҲ¶пјҢеҪ“ж•°жҚ®йңҖиҰҒиў«дёҘж јжҺ§еҲ¶еңЁе®ўжҲ·з«Ҝж—¶пјҢдёҚйңҖиҰҒжҢҒз»ӯзҡ„е°Ҷж•°жҚ®еҸ‘еӣһжңҚеҠЎеҷЁгҖӮ

WebStorageдёӨдёӘдё»иҰҒзӣ®ж ҮпјҡпјҲ1пјүжҸҗдҫӣдёҖз§ҚеңЁcookieд№ӢеӨ–еӯҳеӮЁдјҡиҜқж•°жҚ®зҡ„и·Ҝеҫ„гҖӮпјҲ2пјүжҸҗдҫӣдёҖз§ҚеӯҳеӮЁеӨ§йҮҸеҸҜд»Ҙи·ЁдјҡиҜқеӯҳеңЁзҡ„ж•°жҚ®зҡ„жңәеҲ¶гҖӮ

HTML5зҡ„WebStorageжҸҗдҫӣдәҶдёӨз§ҚAPIпјҡlocalStorageпјҲжң¬ең°еӯҳеӮЁпјүе’ҢsessionStorageпјҲдјҡиҜқеӯҳеӮЁпјүгҖӮ

  • еЈ°жҳҺе‘Ёжңҹ

    localStorage:

    з”ҹе‘Ҫе‘ЁжңҹжҳҜж°ёд№…зҡ„пјҢе…ій—ӯйЎөйқўжҲ–жөҸи§ҲеҷЁд№ӢеҗҺlocalStorageдёӯзҡ„ж•°жҚ®д№ҹдёҚдјҡж¶ҲеӨұгҖӮlocalStorageйҷӨйқһдё»еҠЁеҲ йҷӨж•°жҚ®пјҢеҗҰеҲҷж•°жҚ®ж°ёиҝңдёҚдјҡж¶ҲеӨұгҖӮ

    sessionStorageпјҡ

    з”ҹе‘Ҫе‘ЁжңҹжҳҜеңЁд»…еңЁеҪ“еүҚдјҡиҜқдёӢжңүж•ҲгҖӮsessionStorageеј•е…ҘдәҶдёҖдёӘвҖңжөҸи§ҲеҷЁзӘ—еҸЈвҖқзҡ„жҰӮеҝөпјҢsessionStorageжҳҜеңЁеҗҢжәҗзҡ„зӘ—еҸЈдёӯе§Ӣз»ҲеӯҳеңЁзҡ„ж•°жҚ®гҖӮеҸӘиҰҒиҝҷдёӘжөҸи§ҲеҷЁзӘ—еҸЈжІЎжңүе…ій—ӯпјҢеҚідҪҝеҲ·ж–°йЎөйқўжҲ–иҖ…иҝӣе…ҘеҗҢжәҗеҸҰдёҖдёӘйЎөйқўпјҢж•°жҚ®дҫқз„¶еӯҳеңЁгҖӮдҪҶжҳҜsessionStorageеңЁе…ій—ӯдәҶжөҸи§ҲеҷЁзӘ—еҸЈеҗҺе°ұдјҡиў«й”ҖжҜҒгҖӮеҗҢж—¶зӢ¬з«Ӣзҡ„жү“ејҖеҗҢдёҖдёӘзӘ—еҸЈеҗҢдёҖдёӘйЎөйқўпјҢsessionStorageд№ҹжҳҜдёҚдёҖж ·зҡ„гҖӮ

  • еӯҳеӮЁеӨ§е°Ҹ

    localStorageе’ҢsessionStorageзҡ„еӯҳеӮЁж•°жҚ®еӨ§е°ҸдёҖиҲ¬йғҪжҳҜпјҡ5MB

  • еӯҳеӮЁдҪҚзҪ®

    localStorageе’ҢsessionStorageйғҪдҝқеӯҳеңЁе®ўжҲ·з«ҜпјҢдёҚдёҺжңҚеҠЎеҷЁиҝӣиЎҢдәӨдә’йҖҡдҝЎгҖӮ

  • еӯҳеӮЁеҶ…е®№зұ»еһӢ

    localStorageе’ҢsessionStorageеҸӘиғҪеӯҳеӮЁеӯ—з¬ҰдёІзұ»еһӢпјҢеҜ№дәҺеӨҚжқӮзҡ„еҜ№иұЎеҸҜд»ҘдҪҝз”ЁECMAScriptжҸҗдҫӣзҡ„JSONеҜ№иұЎзҡ„stringifyе’ҢparseжқҘеӨ„зҗҶ

  • иҺ·еҸ–ж–№ејҸ

    localStorageпјҡwindow.localStorage

    sessionStorageпјҡwindow.sessionStorage

  • еә”з”ЁеңәжҷҜ

    localStorageseпјҡеёёз”ЁдәҺй•ҝжңҹзҷ»еҪ•пјҲ+еҲӨж–ӯз”ЁжҲ·жҳҜеҗҰе·Ізҷ»еҪ•пјүпјҢйҖӮеҗҲй•ҝжңҹдҝқеӯҳеңЁжң¬ең°зҡ„ж•°жҚ®

# 2.1 WebStorageзҡ„дјҳзӮ№

пјҲ1пјүеӯҳеӮЁз©әй—ҙжӣҙеӨ§пјҡcookieдёә4KBпјҢиҖҢWebStorageжҳҜ5MBпјӣ

пјҲ2пјүиҠӮзңҒзҪ‘з»ңжөҒйҮҸпјҡWebStorageдёҚдјҡдј йҖҒеҲ°жңҚеҠЎеҷЁпјҢеӯҳеӮЁеңЁжң¬ең°зҡ„ж•°жҚ®еҸҜд»ҘзӣҙжҺҘиҺ·еҸ–пјҢд№ҹдёҚдјҡеғҸcookieдёҖж ·зҫҺиҜҚиҜ·жұӮйғҪдјҡдј йҖҒеҲ°жңҚеҠЎеҷЁпјҢжүҖд»ҘеҮҸе°‘дәҶе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁз«Ҝзҡ„дәӨдә’пјҢиҠӮзңҒдәҶзҪ‘з»ңжөҒйҮҸпјӣ

пјҲ3пјүеҜ№дәҺйӮЈз§ҚеҸӘйңҖиҰҒеңЁз”ЁжҲ·жөҸи§ҲдёҖз»„йЎөйқўжңҹй—ҙдҝқеӯҳиҖҢе…ій—ӯжөҸи§ҲеҷЁеҗҺе°ұеҸҜд»Ҙдёўејғзҡ„ж•°жҚ®пјҢsessionStorageдјҡйқһеёёж–№дҫҝпјӣ

пјҲ4пјүеҝ«йҖҹжҳҫзӨәпјҡжңүзҡ„ж•°жҚ®еӯҳеӮЁеңЁWebStorageдёҠпјҢеҶҚеҠ дёҠжөҸи§ҲеҷЁжң¬иә«зҡ„зј“еӯҳгҖӮиҺ·еҸ–ж•°жҚ®ж—¶еҸҜд»Ҙд»Һжң¬ең°иҺ·еҸ–дјҡжҜ”д»ҺжңҚеҠЎеҷЁз«ҜиҺ·еҸ–еҝ«еҫ—еӨҡпјҢжүҖд»ҘйҖҹеәҰжӣҙеҝ«пјӣ

пјҲ5пјүе®үе…ЁжҖ§пјҡWebStorageдёҚдјҡйҡҸзқҖHTTP headerеҸ‘йҖҒеҲ°жңҚеҠЎеҷЁз«ҜпјҢжүҖд»Ҙе®үе…ЁжҖ§зӣёеҜ№дәҺcookieжқҘиҜҙжҜ”иҫғй«ҳдёҖдәӣпјҢдёҚдјҡжӢ…еҝғжҲӘиҺ·пјҢдҪҶжҳҜд»Қз„¶еӯҳеңЁдјӘйҖ й—®йўҳпјӣ

пјҲ6пјүWebStorageжҸҗдҫӣдәҶдёҖдәӣж–№жі•пјҢж•°жҚ®ж“ҚдҪңжҜ”cookieж–№дҫҝпјӣ

setItem (key, value) вҖ”вҖ” дҝқеӯҳж•°жҚ®пјҢд»Ҙй”®еҖјеҜ№зҡ„ж–№ејҸеӮЁеӯҳдҝЎжҒҜгҖӮ

вҖӢ гҖҖгҖҖ getItem (key) вҖ”вҖ” иҺ·еҸ–ж•°жҚ®пјҢе°Ҷй”®еҖјдј е…ҘпјҢеҚіеҸҜиҺ·еҸ–еҲ°еҜ№еә”зҡ„valueеҖјгҖӮ

вҖӢ гҖҖгҖҖremoveItem (key) вҖ”вҖ” еҲ йҷӨеҚ•дёӘж•°жҚ®пјҢж №жҚ®й”®еҖјз§»йҷӨеҜ№еә”зҡ„дҝЎжҒҜгҖӮ

вҖӢ гҖҖгҖҖclear () вҖ”вҖ” еҲ йҷӨжүҖжңүзҡ„ж•°жҚ®

вҖӢ гҖҖгҖҖkey (index) вҖ”вҖ” иҺ·еҸ–жҹҗдёӘзҙўеј•зҡ„key

# 2.2 cookieгҖҒlocalStorageгҖҒsessionStorageеҢәеҲ«
